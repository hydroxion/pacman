
#### [Pacman](https://github.com/Sphinxs/Pacman)

:fish_cake: Pacman in C++ using Allegro 5

#### About

Pacman, is an arcade game considered one of the classics of the medium, and an icon of 1980s popular culture.

#### Run

```shell
    make && make run
```

#### Debug

```shell
    gdb pacman

    run

    backtrace

    kill
```
